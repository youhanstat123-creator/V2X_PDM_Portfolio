#  V2X PDM

데이터 분석 및 시각화를 기반으로 한 V2X 예지보전 프로젝트입니다.

---

##  프로젝트 소개
V2X 데이터를 기반으로 차량 및 시스템 상태를 분석하고,  
그래프 형태로 시각화하여 상태 변화를 확인할 수 있도록 구현한 프로젝트입니다.

---

##  사용 기술
- Frontend: Vue.js
- Backend: Java, Spring Boot
- Data Processing: Python
- Database: (사용 DB 입력)

---

##  주요 기능

###  데이터 시각화
- 시간별 데이터 그래프 출력
- 상태 변화 추이 확인
- 데이터 기반 시각적 분석 제공

###  데이터 처리
- Python을 활용한 데이터 가공
- 백엔드와 연동하여 데이터 전달

##  담당 역할
- 프론트엔드 화면 구성 및 그래프 UI 구현
- Vue.js 기반 데이터 시각화 화면 구현
- 백엔드 API 연동을 통한 데이터 출력 처리
- 그래프 데이터 변경에 따른 UI 업데이트 처리

---

##  문제 해결 경험
- 그래프 데이터가 정상적으로 출력되지 않는 문제  
  → API 응답 구조 확인 및 데이터 바인딩 수정으로 해결

- 실시간 데이터 반영 문제  
  → 상태 업데이트 로직 개선으로 해결
